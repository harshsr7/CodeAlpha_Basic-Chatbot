Basic ChatBot
Overview
A rule-based chatbot developed in Python that supports English, Hindi, and mixed Hindi-English conversations.

Features
Greetings support
Time, Date, and Day information
Programming-related answers
AI and Machine Learning basics
Simple Math calculations
Jokes and Motivation
Help menu
Exit commands (bye, exit, quit)
Technologies Used
Python 3
datetime module
random module
How to Run
Save the code as smart_chatbot.py

Open Terminal/Command Prompt

Run:

python Basic_chatbot.py
Start chatting with the bot.
